# Military Vocabulary Quiz - GitHub Pages

Statische GitHub Pages-versie van een militaire vocabulairetoets gebaseerd op pagina 4 t/m 16 van de PDF.

## Wat deze versie doet

- student vult naam, klas en eventueel studentnummer in
- meerkeuzevragen Engels -> Nederlands
- automatisch nakijken in de browser
- direct resultaat in percentages
- onder het resultaat een compleet antwoordenoverzicht
- lokale opslag van eerdere resultaten op hetzelfde apparaat
- zinnen met dikgedrukte woorden of woordgroepen die vertaald moeten worden
- visuele ondersteuning met afbeeldingen uit de bron-PDF

## Beperkingen van GitHub Pages

Deze versie heeft geen backend. Resultaten worden alleen lokaal in de browser opgeslagen.

## Publiceren op GitHub Pages

1. Maak een nieuwe repository op GitHub.
2. Upload alle bestanden uit deze map.
3. Ga naar **Settings** -> **Pages**.
4. Kies de branch **main** en de map **/(root)**.
5. Sla op.
6. Na enkele minuten staat de site live op je GitHub Pages URL.

## Bestanden

- `index.html` - paginaopbouw
- `styles.css` - vormgeving
- `questions.js` - alle toetsvragen
- `app.js` - inloggen, nakijken en resultaten tonen
- `assets/pages/` - afbeeldingen uit de PDF
